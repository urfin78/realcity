# Kartendaten — Lizenzhinweis

Die Dateien `maps/*.json` in diesem Verzeichnis sind **abgeleitete Datenbanken**
im Sinne der Open Database License (ODbL) und kombinieren zwei Quellen:

| Anteil | Quelle | Lizenz |
|--------|--------|--------|
| Wasser, Wald, weitere Vektordaten | © OpenStreetMap-Mitwirkende | **ODbL 1.0** |
| Höhendaten (elevation) | SRTM (NASA / CGIAR-CSI) | Public Domain |

Weil OSM-Daten enthalten sind, gilt für die **Gesamtdatei** die ODbL:

- **Attribution** — bei jeder Nutzung/Anzeige muss „© OpenStreetMap-Mitwirkende"
  genannt werden (im Spiel geschieht das in `index.html`, Element `#attribution`).
- **Share-Alike** — wer diese Dateien (oder eine veränderte Fassung) öffentlich
  weitergibt, muss sie ebenfalls unter ODbL anbieten.

Diese Lizenzbedingung ist **unabhängig** von der MIT-Lizenz des Quellcodes
(siehe `../LICENSE`). Der Code ist MIT, die Kartendaten sind ODbL.
